## Interactive mode
* Touch mode
* Gyroscope mode

## DisplayMode
* Glass mode
* Eye mode

## 基于以下开源项目实现
* IJKPlayer
* MD360player4Android 
* Glide

## 更新日志

## 你可以在这个项目中学到
* Databinding
* 沉浸式状态栏实现
* 一种优雅的视频播放器实现，就像原生的ViedoView和Mediacontroller
* 一种值得参考的VR播放器的实现方式
